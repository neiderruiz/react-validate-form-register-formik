# Validate form register with React (Formik, Yup) and Vite

## clone project

```bash
git clone https://github.com/neiderruiz/react-validate-form-register-formik.git
```

```bash
cd react-validate-form-register-formik
```

## Project setup

```bash
npm install
```

## run project

```bash
npm run dev
```
